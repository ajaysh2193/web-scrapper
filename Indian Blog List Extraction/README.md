A Simple beautifulsoup-enabled Py script to get the list of blog urls from (https://indianbloggers.org/) and outputing a csv with the list. 

* BeautifulSoup (Extraction)
* Pandas (Output File)
* Matplotlib (Blog Hosting Type Plot)
